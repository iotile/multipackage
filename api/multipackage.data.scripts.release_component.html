multipackage.data.scripts.release_component.parse_version(version_line, prefix='##')[source]¶

Parse a version from a markdown header.

The line must be formatted as: ## X.Y.Z [whatever else you want]

So you could have, for example: ## 1.5.0 (11/28/2018) ##1.5.0

## 1.5.0 hello this is a test

Returns:The parsed version.
Return type:str
